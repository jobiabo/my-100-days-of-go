# Exercise 2: Personalized Resume Summary
#### Objective: Apply prompting to professional use cases.
### Write a prompt that generates a resume summary based on role and skills. Example: "Create a 3-sentence professional summary for a software engineer skilled in Python and cloud systems.
### Adjust the prompt for a different role (e.g., teacher, designer, marketer).
### Compare how the outputs change depending on the role.

* **Model Used:** [OpenAI](https://www.chatgpt.com)

* **Prompt 1:** Create a 3-sentence professional summary for a software engineer skilled in Python and cloud systems.
* **Model Output:** Results-driven software engineer with extensive experience in Python development and designing scalable cloud-based solutions. Proficient in leveraging cloud platforms to build, deploy, and optimize high-performance applications. Adept at collaborating with cross-functional teams to deliver robust, efficient, and secure software solutions that meet business objectives.

* **Prompt 2:** Write a 3-sentence professional summary for an AI engineer skilled in Python, TensorFlow, and cloud-based AI solutions.

* **Model Output:** Versatile AI engineer with expertise in Python and TensorFlow, specializing in developing advanced machine learning and deep learning models. Experienced in deploying and optimizing cloud-based AI solutions to deliver scalable, high-performance applications. Adept at translating complex data into actionable insights, collaborating with cross-functional teams, and driving innovation in AI-driven products.

# Comparison 

The model was able to detect the roles and suit every ideas and logic associated to that role.
