# 🚀 My 100 Days of Go

 *A living repository documenting my journey of learning Go through consistent practice, experimentation, and project development.*

---

## 📖 About

Welcome!

This repository is my personal Go (Golang) practice space where I experiment with ideas, rebuild projects, solve programming exercises, and strengthen my understanding of software engineering.

Unlike a traditional tutorial repository, this workspace evolves naturally as I learn. Some directories contain complete projects, while others are dedicated to experimenting with new concepts, refactoring old code, or rebuilding previous work with better techniques.

Every commit represents another step in becoming a better developer.

---

## 🎯 Objectives

* Learn Go by building real programs
* Practice writing clean and maintainable code
* Reinforce programming fundamentals through repetition
* Improve problem-solving skills
* Experiment with new ideas without fear of making mistakes
* Build a strong foundation for backend development

## 💡 My Learning Philosophy

I don't measure progress by the number of tutorials I complete.

I measure it by:

* Writing code
* Breaking things
* Fixing them
* Rebuilding them
* Understanding why they work

Many folders contain multiple versions of the same project because improvement comes from iteration. Every rewrite helps reinforce concepts, expose better design choices, and deepen my understanding of Go.
